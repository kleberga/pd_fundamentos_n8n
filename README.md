## PD da disciplina "Fundamentos de Workflows no n8n [26E1_2]"

Foi desenvolvido um aplicação na plataforma n8n para automatizar a captura de notícias sobre um assunto, realizar um resumo das notícias e encaminhar esse resumo para o e-mail e o Google Drive.
Os inputs fornecidos pelo usuário são:
a) o assunto a ser pesquisado;
b) a data de início e de fim de publicação das notícias;
c) se a busca deve ser na internet, no banco de dados vetorial ou em ambas; e
d) o e-mail para envio do resumo.
